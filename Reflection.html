My Reflection

What did you learn this Quarter?

I've realized how much ICT, especially coding, has influenced both my own skills and how I perceive the world. Each line of code is a component that fits into a bigger image, making coding similar to a puzzle. There is a great sense of purpose that comes from using programming to solve issues. It's not only about writing code, I've realized; it's also about knowing how things operate and being able to build anything from the ground up. The difficulties first appeared to be too much to handle, particularly when I encountered errors or was unable to determine why my code wasn't functioning. But with every mistake, I learned something new, and that sense of growth kept me going. The fact that a few lines of code can be a tool that unlocks a wide range of opportunities is amazing. Patience, logical reasoning, and the ability to deconstruct complicated issues into smaller ones are all lessons learned during the process. Looking ahead, I realize how essential coding is in nearly every field, and it excites me to think about where it can take me in my future. I'm becoming more and more aware of how much I don't know, but it just makes me want to learn more.
